# Neural Ordinary Differential Equations

## Overview

Implementation of Neural Ordinary Differential Equations (Neural ODEs), a class of deep neural network models that use ODE solvers as part of the network architecture.

## Paper Reference

**Neural Ordinary Differential Equations**  
Authors: Ricky T. Q. Chen, Yulia Rubanova, Jesse Bettencourt, David Duvenaud  
Conference: NeurIPS 2018  
Paper: [arXiv:1806.07366](https://arxiv.org/abs/1806.07366)

- `docs/` - Documentation and notes
- `data/` - Datasets used for experiments
- `scripts/` - Implementation scripts
- `results/` - Experimental results and visualizations
